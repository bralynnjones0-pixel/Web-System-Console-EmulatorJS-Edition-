Web System Console (EmulatorJS Edition)
web-based emulator console that supports multiple retro systems. This project uses EmulatorJS for the emulation cores and IndexedDB to keep your game library saved locally in your browser.
Features
Local Library: ROMs you upload stay in your browser even after you refresh.
Wide Support: - Nintendo DS (.nds)
PSP (.iso, .cso)
GameBoy Advance/Color (.gba, .gb)
NES/SNES (.nes, .smc)
N64, PS1, and more.
How to Use
Open the App: Open system_console.html in any modern web browser.
Load Games: Click the "LOAD NEW ROMS" button in the sidebar to add your .nds, .gba, or other game files.
Play: Click a game title in the sidebar to boot the emulator.
Tech Stack
HTML5/CSS3: Modern dark-mode UI.
JavaScript: Custom library management and emulator integration.
IndexedDB: Persistent local storage for ROM files.
EmulatorJS: Core emulation engine.
